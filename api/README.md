API for Native Apps

for auto trading

1. Get Firebase AccessToken from Native apps
2. Request User Exchange Info to Firebase by using AccessToken
3. Order crypto assets by using User Exchange Info

## APIs

- Save User Info
- Save Exchange Info
- Save Auto Trading Info
- Auto Trading

User Info
